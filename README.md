# Mask Databases

Natural language database layer for Node.js and TypeScript. Write prompts in code; Mask compiles them into real database queries at build time and runs them with zero overhead at runtime.

**Full documentation:** [maskdatabases.com/docs](https://maskdatabases.com/docs)

---

## Install

```bash
npm install mask-databases@beta
```

## Supported Databases

MongoDB, Mongoose, MySQL, MariaDB, PostgreSQL, SQLite, Oracle, Neo4j
